# Modbus-Excel
A tool to read input registers in real-time from Excel (xlsx)

Default Configuration Is: 

method:rtu 

port:COM6 

timeout:10(ds) 

baudrate:19200 

stopbits:1 

parity:Even 

bytesize:8 

number_of_holding_registers:8

number_of_input_registers:8

number_of_coils:8

address:0x00
  
Refresh time is 1 second
